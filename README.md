# sdo-transfer-helper
Tampermonkey-скрипт для сравнения дисциплин и подготовки перезачёта

## Использование

1. Откройте страницу оценок
2. Нажмите «1. Сохранить оценки»
3. Откройте пройденный курс
4. Нажмите «2. Сохранить старый курс»
5. Откройте актуальный курс
6. Нажмите «3. Сравнить с актуальным»
8. Отчет готов

Перед сравнением другого студента или другой пары курсов нажмите «Сбросить данные»
